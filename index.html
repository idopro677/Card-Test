♠ Features Guide

📷

Live Mode

Opens your camera and runs YOLOv8 on every frame for real-time card detection. Bounding boxes are color-coded by suit: black for Spades, red for Hearts, blue for Diamonds, green for Clubs.

📸

Photo Mode

Upload an image from your gallery. The AI instantly detects all cards and draws bounding boxes with labels and confidence scores.

⏳

Countdown Freeze

When ON in Live mode, pressing Freeze shows a 3-2-1 countdown with audio ticks before capturing, giving you time to steady the camera.

🎯

Confidence Slider

Sets the minimum detection confidence. Lower = more detections; higher = fewer but more accurate. Default 45%.

🃏

Blackjack Advisor

Input your hand and the dealer's up card by taking photos or selecting from the card picker. The AI uses Hi-Lo card counting + basic strategy to recommend the ideal move.

📊

Hi-Lo Count

The running count tracks 2–6 as +1, 7–9 as 0, and 10–A as -1. The true count = running ÷ decks remaining. A high positive count favors the player.

👟

Shoe Size

Select how many decks are in the shoe (1–8). This is used for true count calculation and probability accuracy. Typical casino shoes are 6 or 8 decks.

🔄

New Hand / Reset

New Hand clears the current hand but preserves the running count. Reset Count clears the count entirely (use when a new shoe starts).

🏳️

Surrender Option

Toggle Surrender on if the casino allows it. The advisor will recommend surrender when mathematically optimal (e.g., hard 16 vs dealer 9/10/A).

♣

Texas Hold'em Advisor

Input your hole cards and community cards (Flop/Turn/River). The AI calculates hand probabilities and recommends the optimal action based on pot equity.

🔥

Burn Cards

Optionally enter burned cards to improve probability calculations. Removing known cards from the deck increases the accuracy of outs calculations.

📈

Outs & Probabilities

At each street, see the probability of hitting each hand type, your total outs, and the pot odds break-even percentage needed to call profitably.

🔄

Per-Turn Reset

Poker uses a single deck per game. Stats reset each new hand. Use "Next Street" to advance from Preflop → Flop → Turn → River.
